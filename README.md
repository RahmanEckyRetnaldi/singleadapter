Cara install

# Tambahkan JitPack repository ke build file
```
	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
# Tambahkan dependency
```
dependencies {
	        implementation 'com.github.triandamai:responseconverter:v1.0.4'
	}
```
